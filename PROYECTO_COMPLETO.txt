📊 EJEMPLOS DE CONSULTAS API
═════════════════════════════════════════════════════════════════════════════════

GET /countries
  Retorna: Lista de todos los países

GET /rules/1
  Retorna: Reglas del país con ID 1 (Honduras)

GET /statistics?country_id=2
  Retorna: Estadísticas del país con ID 2 (México)

GET /history?page=1
  Retorna: Historial paginado (15 registros/página), total de registros y página actual

GET /report/5
  Retorna: Reporte completo de evaluación con ID 5

POST /validate
  Parámetros: country_id (form), file (multipart)
  Retorna: Resultados de validación completos
